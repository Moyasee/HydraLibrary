Suggest a Source

Want to add a new source to Hydra Library? You can suggest it by creating an issue on our GitHub repository.

Important Requirements:

  • Only .json file links are accepted (e.g., repack-games.com/source.json)
  • Website links are not accepted (e.g., repack-games.com)

How to Suggest a Source:

  1. Visit our GitHub repository issue page
  2. Click on "New Issue"
  3. Select "Source Suggestion" template
  4. Fill in the required information:
    • Source URL (.json file)
    • Source Name
    • Description
    • Approximate number of games
  5. Submit the issue
