About Hydra Library

Hydra Library is a comprehensive game source management system designed to help users discover, organize, and access various game repositories safely and efficiently.

Source Status Types:

  • Trusted - Verified and safe sources
  • Safe For Use - Community-verified sources
  • Use At Your Own Risk - Unverified sources
  • New - Recently added sources

Features:

  • Advanced filtering system
  • Source status indicators
  • Game count tracking
  • Safe copy and install options
  • Risk warnings for unverified sources

Suggest a Source

Want to add a new source to Hydra Library? You can suggest it by creating an issue on our GitHub repository. Here's how:

How to Suggest a Source:

  1. Visit our GitHub repository issue page
  2. Click on "New Issue"
  3. Select "Source Suggestion" template
  4. Fill in the required information:
    • Source name
    • Source description
    • Source URL
    • Approximate number of games
    • Any additional information
